## Dataset
The dataset used in this project is the Telco Customer Churn dataset.
It was obtained from Kaggle, which is one of the approved open-data sources listed in the Lab 2 instructions.

## Machine Learning Problem
The goal of this project is to predict whether a customer will churn or not based on customer-related features such as contract type, payment method, and services used.

## Problem Type
This is a classification problem.

## Target Variable
The target variable in this dataset is **Churn**, which indicates whether a customer has left the company or not.
